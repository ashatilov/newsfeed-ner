Суть

Проект представляет собой агрегатор новостных лент за текущие сутки c распознаванием именованных сущностей в заголовках по следующим категориям:

  • персоны
  • локации
  • организации

Источники

Новостные источники взяты в основном из рейтинга самых цитируемых СМИ за январь 2019 на сайте Медиалогии, который был дополнен несколькими информационными агентствами со страницы в википедии.

Как работает

Сбор информации

Практически все источники имеют RSS каналы, поэтому почти вся информация собирается с помощью библиотеки feedparser.
Был написан один отдельный парсер (requests + Beautiful soup) для самого цитируемого (судя по рейтингу) информагентства «РИА Новости», т.к. у него нет RSS канала.

Распознавание именованных сущностей

Именованные сущности распознаются с помощью модели для русского языка библиотеки DeepPavlov, которая показывает отличный результат «из коробки». Подробнее модель описана в документации и статье Application of a Hybrid Bi-LSTM-CRF model to the task of Russian Named Entity Recognition.

Нормализация сущностей

Нормализация производится с помощью pymorphy2.

Сайт

Сайт реализован на Django и PostgreSQL, оформление Bootstrap, облака слов рисуются в wordcloud.
